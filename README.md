## 三行指令，即刻入島

```bash
git clone https://github.com/eric6300/greed-island
cd greed-island
claude
```

輸入 `/start`。Eta 會在紫蘇之樹迎接你。

## 這是什麼？

貪婪之島是一款**完全在 Claude Code 中運行的文字冒險遊戲**。沒有安裝步驟、沒有外部依賴、不需要任何程式語言。你只需要 `git` 和 [Claude Code](https://claude.ai/code)。

Claude 扮演遊戲主持人（GM），以沉浸式的文字敘事主持整個世界。你探索島嶼、蒐集卡牌、訓練念能力、結交盟友或樹立敵人——你的每個選擇都會改變這座島的命運。

## 遊戲特色

### 忠實還原的貪婪之島

100 張限定槽卡、40 張咒語卡、12 座城市與地區。從安托基巴的月賽到索夫拉比的躲避球決賽，從馬薩多拉的咒語卡商店到多利亞斯的致命賭局——原著中的一切都在這裡等你。

### 完整的念能力系統

水見式決定你的念系。纏、絕、練、發、凝、周、圓、堅、硬、隱、流——十一種念技法隨等級逐步解鎖。設計你專屬的必殺技，在誓約與力量之間找到平衡。

### Roguelike：死亡是永久的

沒有讀檔。沒有重來。HP 歸零的那一刻，這個周目就結束了。但死亡不是終點——每次通關（或死亡）都會解鎖新的知識，讓你在下一個周目走上截然不同的道路。

### 這座島有自己的意志

你以為你能改變一切？試試看。

### NPC 會記住你

比司吉記得你每一次訓練的表現，態度會隨你的努力而軟化。西索在暗處觀察你，興趣隨你的實力增長。磊札尊重正面挑戰的勇者。甘舒⋯⋯正在評估你是否值得先發制人。

這些記憶跨越對話、跨越 session，他們是活的。

### 多周目：每一輪都是新的故事

通關後帶出 3 張卡牌，下一輪以全部實力重新開始。你在這一輪目睹的悲劇、錯過的機會、來不及做出的選擇——下一輪，你會知道該怎麼做。

每次通關都解鎖新的可能性。有些路，只有經歷過失敗的人才看得見。

## 指令

在遊戲中，你可以使用以下指令：

| 指令 | 說明 |
|------|------|
| `/start` | 開始新遊戲，或從上次離開的地方繼續 |
| `/book` | 召喚咒語書，查看所有卡牌蒐集進度 |
| `/look` | 仔細觀察你所在的場景 |
| `/go [地點]` | 前往另一個地點 |
| `/talk [NPC]` | 與角色交談 |
| `/gain` | 將擊敗的敵人或發現的物品轉為卡牌 |
| `/cards` | 快速確認持有卡牌數量 |
| `/use [卡牌]` | 發動卡牌的效果 |
| `/transform` | 將卡牌轉換為實體物品 |
| `/slotin [卡牌]` | 將卡牌放入限定槽 |
| `/clone [卡牌] [隊友]` | 複製一張卡牌給隊友 |
| `/memcard` | 查看你的記憶卡——通關紀錄、傳家寶、未解鎖結局 |

你也可以直接說出 **Book** 或 **Gain**，GM 會將它視為遊戲內的咒語。

## FAQ

**Q: 我不認識 Hunter × Hunter 可以玩嗎？**
A: 可以。GM 會在遊戲中自然地解釋所有概念。你不需要任何前置知識。

**Q: 一輪大概要玩多久？**
A: 取決於你的探索深度。直衝主線可能數小時，完整探索需要更長時間。

**Q: 遊戲支援英文嗎？**
A: GM 會自動偵測你使用的語言並以相同語言回應。用英文對話，GM 就用英文敘事。

**Q: 死了真的不能讀檔？**
A: 真的。但你的知識和傳家寶會帶到下一輪。每一次死亡都讓你更強。
