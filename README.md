## 三行指令，即刻入島

```bash
git clone https://github.com/eric6300/greed-island
cd greed-island
claude
```

輸入 `/start`。剩下的，島上會告訴你。

---

## 這是什麼？

貪婪之島是一款**完全在 Claude Code 中運行的文字冒險遊戲**。沒有安裝步驟、沒有外部依賴、不需要任何程式語言。你只需要 `git` 和 [Claude Code](https://claude.ai/code)。

Claude 扮演遊戲主持人（GM），以沉浸式的文字敘事主持整個世界。探索島嶼、蒐集卡牌、訓練念能力、結交盟友或樹立敵人——你的每個選擇都有後果。

---

## 遊戲特色

### 忠實還原的貪婪之島

城市、荒野、港口、賭場——原著中的地點都在這裡。卡牌散落在世界各處，每一張都有獨特的取得條件。有些需要戰鬥，有些需要智慧，有些需要你付出意想不到的代價。

### 完整的念能力系統

水見式決定你的念系。隨著冒險的深入，你將逐步解鎖更強大的念技法，設計你專屬的必殺技。誓約越重，力量越強——但代價也越大。

### Roguelike：死亡是永久的

沒有讀檔。沒有重來。HP 歸零的那一刻，一切結束。

但結束不代表白費。

### 這座島有自己的意志

你以為你能改變一切？試試看。

### 他們是活的

島上的角色會記住你。你的每一個行為，都在改變他們對你的看法。

## 基本指令

| 指令 | 說明 |
|------|------|
| `/start` | 開始冒險 |
| `/look` | 觀察周圍 |
| `/go [地點]` | 前往某地 |
| `/talk [角色]` | 與人交談 |
| `/book` | 打開你的咒語書 |

更多指令在遊戲中自然學會。你也可以直接說出「**Book**」或「**Gain**」——GM 會將它視為咒語。
