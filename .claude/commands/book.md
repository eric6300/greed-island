# /book — 召喚咒語書

召喚 Book（咒語書），顯示所有卡牌蒐集狀態。口語說「Book」也觸發此指令。

## 執行流程

1. 委託 game-engine 讀取 game-state.json 的 cards 區段
2. 以以下格式顯示：

```
📖 BOOK

【限定槽（Specified Slots）】
000 ░░░░░░░░░   001 ░░░░░░░░░   002 ░░░░░░░░░
003 ░░░░░░░░░   004 ░░░░░░░░░   005 ░░░░░░░░░
...
（已取得的顯示卡牌名稱，未取得的顯示 ░░░░░░░░░）
（共 100 格，每行 3 張，#000 特別標記）

【自由槽（Free Slots）】
[1] [卡牌名稱]（[類型]，Rank [等級]）
[2] [卡牌名稱]（[類型]，Rank [等級]）
...
（最多 45 格）

進度：[specified_count]/99（#000 另計）
自由槽：[free_slots count]/45
```

3. 若有傳家寶卡牌，在自由槽中標記「⭐ 傳家寶」
4. 描述咒語書從空氣中浮現的視覺效果
