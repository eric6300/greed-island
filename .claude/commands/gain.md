# /gain — 撿取卡牌

撿取場景中可用的卡牌。口語說「Gain」也觸發此指令。

## 執行流程

1. 委託 game-engine 讀取 game-state.json（cards）

2. **檢查是否有可撿取的卡牌**：
   GM 內部追蹤「當前場景是否有掉落的卡牌」（來自戰鬥勝利、解謎完成等）。

3. **有卡牌可撿取**：
   - 檢查是否在 1 個行動的時限內（上一個行動觸發了卡牌掉落）
   - 若超時：「卡牌已經消失了⋯⋯散發出微光後化為虛無。那個物品永遠無法再轉為卡牌了。」
   - 若及時：
     a. 判斷卡牌類型
     b. 若為限定槽卡 → 自動放入對應限定槽，specified_count +1
     c. 若為自由卡/咒語卡 → 放入 free_slots
     d. 若 free_slots 已滿（45 張）→ 詢問玩家要丟棄哪張
   - 描述卡牌收入的效果：物品化為光芒，凝聚成一張閃爍的卡牌，滑入你的 Book

4. **沒有卡牌可撿取**：
   「周圍沒有可以撿取的卡牌。」

5. 更新 action_count +1
6. 委託 game-engine 寫入 game-state.json

## 重要規則
- Gain 必須在卡牌掉落後的下一個行動內執行
- 若玩家在掉落後先執行了其他行動（/go、/look 等），再說 Gain → 超時
- 超時的物品永遠無法再轉為卡牌（在 game-state 中記錄已消失的卡牌）
