# /look — 查看場景

查看當前所在地點的詳細描述。

## 執行流程

1. 委託 game-engine 讀取 game-state.json 的 location
2. 使用 /location-info [當前地點] 取得地點詳細資料
3. 使用 /npc-profile 確認該地點當前有哪些 NPC（根據遊戲階段）

4. 以沉浸式語氣描述：
   - 環境氛圍（天氣、光線、聲音、氣味）
   - 地標或建築
   - 可見的 NPC（列出名字和簡短描述）
   - 可互動的物品或地點
   - 出口方向（可前往的連接地點）

5. 不顯示數值，純敘事描述

6. 結尾提供選項提示（不超過 4 個）
