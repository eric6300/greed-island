查詢戰鬥規則與判定方式。此為 GM 內部查詢工具，玩家不可見。

用法：
  /combat-rules              → 回傳核心戰鬥規則摘要
  /combat-rules 流           → 查詢流（Ryu）的規則
  /combat-rules 特殊怪物     → 查詢特殊怪物的取得條件
  /combat-rules 體力         → 查詢體力消耗規則

可查詢主題：公式、技法係數、戰鬥結果表、體力消耗、先攻判定、骰子機制、流模式、特殊怪物

執行步驟：
1. 讀取 world/combat_rules.md
2. 若有指定查詢主題，只回傳相關段落
3. 若無指定主題，回傳核心摘要：攻擊公式、防禦公式、結果判定表、骰子機制
4. 格式化為 GM 可直接套用的規則摘要

注意：回傳不超過 800 tokens。若查詢「完整規則」，優先回傳公式和結果表，省略範例。
