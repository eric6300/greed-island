# 貪婪之島 — 卡牌取得規則

## 野外卡牌（怪物/物品）

1. 找到對應地點
2. 完成特定條件（打倒、解謎、對話）
3. 在 1 個行動內說出「Gain」
4. 超時則卡牌消失，該物品**永遠無法**再轉為卡牌

**重要**：Gain 的時限是「下一個行動」。若玩家在卡牌掉落後先執行了其他行動（/go、/look 等），再說 Gain → 超時失敗。

## Antokiba 月賽

**時間**：每月 15 日（遊戲內），玩家在 Antokiba 時觸發
**類型**（GM 隨機決定）：
- **問答型**：GM 出題（貪婪之島相關知識），玩家直接回答
- **戰鬥型**：使用戰鬥系統判定
- **技藝型**：玩家描述策略，GM 判定創意分

**獎品**：一張限定槽卡（GM 根據當前進度選擇適當的卡）

## Masadora 咒語卡購買

**價格**：10,000 Jenny = 3 張隨機咒語卡
**機制**：GM 用骰子決定（時間戳法，從 40 張中隨機 3 張）
**重複**：可能抽到已持有的咒語卡

## 大天使的氣息 特殊取得

**條件**：在 Masadora 咒語卡商店集齊全 40 種不同咒語卡
**交換**：40 張咒語卡全部回收，換取 大天使的氣息（#017）
**效果**：治癒任何疾病

## 卡牌等級與轉換上限

從低到高：H → G → F → E → D → C → B → A → S → SS

**Conversion Limit**（同時存在於世界中的最大數量）：

| 等級 | 上限 |
|------|------|
| H-G | 100 |
| F-E | 50 |
| D-C | 30 |
| B | 20 |
| A | 10 |
| S | 5 |
| SS | 3 |

等級越高，其他玩家也持有的機率越低，競爭越激烈。
