# 貪婪之島 — 劇情事件資料庫

## EVENT-001：比諾特訓練（耐久成長模式）

**觸發**：進入 Perilla Tree 區域，比司吉已加入隊伍
**時長**：10 天（50 個行動）

### 規則
- 比諾特必須在兩週內閃躲玩家攻擊
- 每天結束時：念等級 +0.5
- 比諾特閃躲品質每 3 天提升 1 級
- **核心限制**：禁止全力硬或高傷害發。一擊打倒 = 訓練中止，失去後續成長

### Day 10 判定
比諾特宣告投降。玩家選擇：
- **放走**：比諾特感動自首，未來可能以 NPC 提供情報。比司吉滿意。
- **擊倒**：比司吉沉默。此後訓練效率 -10%。

### 獎勵
- 念等級 +5（完整完成）
- 解鎖凝的進階應用

## EVENT-002：Bopobo相撲（入場關卡）

**觸發**：第一次前往 Soufrabi，在酒館遇到Bopobo
**規則**：火圈相撲，先踏出圈外者輸。念允許，武器禁止。

### 選項
1. **正面硬推**：體格×2 + 練係數 vs Bopobo體格 28。念 <25 幾乎必敗。
2. **速度閃躲**：速度×3 + 1d10，需連續成功 3 次。
3. **發現弱點（原著解法）**：使用凝觀察 → GM 提示「他對突然的熱有本能反應」→ 利用火/酒精 → 自動勝利。

**失敗**：無懲罰可重來，但Bopobo敵意增加。

## EVENT-003：索夫拉比海盜戰（競技指揮模式）

**觸發**：第二次前往 Soufrabi，帶齊 15 人隊伍

### Phase 1：普通競技（先取 4 勝）
| 項目 | 判定屬性 |
|------|---------|
| 拳擊 | 體格 + 練係數 |
| 相撲 | 體格 + 耐久 |
| 摔角 | 體格 + 速度 |
| 游泳 | 耐久 + 速度 |
| 棒球 | 速度 + 感知 |

偵查策略：刻意輸第一輪 → 記錄對手念能力 → 再戰 +3 加成

### Phase 2：躲避球決賽（8v8，值 8 分）
玩家方 8 人 vs 磊札 + 7 個惡魔

每回合戰術指令：
- [A] 攻擊——投球，選擇目標
- [B] 防守——接球（難度依投球者強度）
- [C] 傳球——+30% 下回合攻擊
- [D] Back 卡——復活已出局隊友（限 3 次）

**磊札的球**：
- 普通防守必然出局
- 用硬擋 → 擋住但自己出局
- 西索橡皮糖接力 → 抓住（消耗橡皮糖）
- 果列奴白猩猩緩衝 → 猩猩代出局

**最終決戰**：玩家必須發動最後一擊。念等級×強化係數 > 磊札防禦 85。

## EVENT-004：甘舒最終戰（策略解謎模式）

**觸發**：持有 99 張限定卡，絕茲絕拉拖延期滿

### 戰前計畫
GM 要求：「你打算怎麼對付甘舒？」

- **A 級計畫**（汽油 + 陷阱地形）：解鎖所有選項，正常難度
- **B 級計畫**（只有其中一項）：部分選項關閉，較高難度
- **無計畫**：特殊選項全關，念 Lv.60 以下幾乎必敗

### 三場同時進行
- 玩家 vs 甘舒（最難）
- 奇犽 vs 沙度（中等）
- 比司吉 vs 巴拉（較易）

玩家選擇打哪一場。其餘由 NPC 代打（非 100% 勝率）。

### 甘舒戰關鍵機制
**小花告知**：甘舒雙手包覆凝 = 小花準備發動
- [1] 犧牲手讓他炸，用流把 70% 氣留在腳踢下巴
- [2] 閃避（速度判定）
- [3] 汽油破解（若有準備）

**倒數炸彈**：若玩家身上有炸彈
- 每行動 -50，戰鬥中 -100/回合
- 絕 = -10/回合
- 解除：觸碰甘舒說「我抓到炸彈魔了」

## EVENT-005：多利亞斯賭城（博弈模式）

**觸發**：進入多利亞斯賭場

### 吃角子老虎
- 普通：骰子判定
- 持有 Risky Dice：1d20，2-20 = 大吉，1 = 骷髏（失去全部 Jenny + 詛咒 5 行動 -5）

### 賭場之王撲克
- 觸發：贏過吃角子一次後
- 策略判定，Risky Dice 提升勝率
- 贏：限定卡 Fledgling Gambler
- 輸：損失賭注

### 上癮機制
- 連續用 Risky Dice 3 次 → 衝動提示
- 繼續 → 成癮旗標（未來賭博場合觸發衝動判定）
- 比司吉在場 → 自動阻止第 4 次

## EVENT-006：通關問答大賽（知識考核模式）

**觸發**：任何玩家蒐集完 99 張限定卡

### 流程
1. Eta全島廣播：「10 分鐘後問答大賽」
2. 10 分鐘準備（可翻 Book，不能用 Analysis）
3. 100 題關於限定卡取得細節

### 計分
- 親自取得的卡 = +1
- 造訪過相關地點 = +0.5
- 純猜測 = +0.25 期望值

### NPC 競爭分數
- 甘舒存活且自由 = 82
- 絕茲絕拉存活 = 75
- 一般玩家 = 60-70

### 結果
- 勝：Ruler's Invitation，前往利梅羅
- 敗：搶奪（體力+速度判定）或等下次機會

## EVENT-007：港務長無限任務（談判解謎模式）

**觸發**：前往貪婪之島港口想離島

### 選項
1. **賄賂**：5,000-50,000 Jenny（骰子決定），太少 → 漲價
2. **打倒**：念 20+ 輕鬆勝，艾蕾娜出現
3. **接受任務（陷阱）**：每完成一個新任務出現 → 用凝 → 發現無終點 → 解鎖：「你看穿了」
4. **Leave 卡**：消耗稀有卡直接離島

## EVENT-008：安托基巴餐廳挑戰（限時小事件）

**觸發**：進入 Caveau de Riquewihr 並點餐

### 挑戰（3 個行動內完成）
1. **專心吃**：耐久判定，失敗 → 1 行動不能戰鬥
2. **周強化消化**：念 20+ 自動成功
3. **分給隊友**：每多一人 -1 行動（最少 1）
4. **跳過**：付錢離開

### 結果
- 1 行動完成：廚師驚訝 + 額外情報
- 2-3 行動：免費 + Galgaida 自由卡
- 失敗：付錢，無獎勵

## EVENT-009：幻影旅團遭遇（極限危機模式）

**觸發**：中期，野外或城市隨機遇到飛坦或乒奇斯

### GM 宣告
「⚠ 你感受到一股令人窒息的殺氣。這絕對不是普通的玩家。」

### 念 < 50
1. 立刻逃跑（速度判定，70%）
2. 裝沒事走開（感知，60%）
3. 主動攻擊（幾乎必敗，打出傷害 → 旅團尊重）

### 念 50-70
1. 逃跑（幾乎必成功）
2. 短暫交手撤退（受傷但存活，+情報）
3. 有旅團聯絡人 → 可談判

### 念 80+
可正式交戰，仍有風險

### 旅團派玩家
成員不攻擊，可能要求協助找 Exorcist 線索。拒絕 → 關係值下降。
