# 貪婪之島 — 傳家寶系統

## 基本規則

上一周目帶出的 3 張卡牌，在新周目開始時成為「傳家寶」：
- 自動出現在 Book 自由槽
- **效果 100%，不弱化**——這是你用一整個周目換來的獎勵
- 不佔用限定槽，可正常使用
- 通關時選擇帶出哪 3 張卡是最重要的戰略決策

## 策略影響範例

| 帶出的卡 | 下一周目效果 |
|---------|------------|
| #017 大天使的氣息 | 完全治癒任何疾病——可以提前救巴特拉的戀人 |
| #039 同行 | 開局直接傳送到任意城市，跳過前期探索 |
| #084 聖騎士的首飾 | 完全抵擋咒語 + 解除手上中咒的卡片，甘舒戰有巨大優勢 |
| #025 風險骰子 | 開局即有賭博利器，多利亞斯速通 |
| #026 聖水 | 防禦攻擊型咒語 10 次 |
| #002 一坪海岸線 | 跳過索夫拉比海盜戰，直接進入海神之家 |
| #081 藍色行星 | 直接完成比司吉的願望支線，好感度 +5 |
| #083 真實之劍 | 一刀砍斷假的東西，審判和戰鬥兩用 |

## 帶出的抉擇

只能帶 3 張——這就是平衡。每次通關都是一次痛苦的選擇：

- **走劇情路線**：大天使的氣息 + 同行 + 情報卡 → 速通救人
- **走戰鬥路線**：真實之劍 + 聖騎士的首飾 + 聖水 → 正面碾壓
- **走探索路線**：一坪海岸線 + 風險骰子 + 藍色行星 → 解鎖隱藏內容
- **走旅團路線**：戰鬥裝備為主 → 應對高強度敵人

## game-state.json 格式

```json
"heirlooms": [
  {
    "name": "大天使的氣息",
    "original_effect": "治癒任何疾病"
  }
]
```
