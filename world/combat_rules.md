# 貪婪之島 — 戰鬥判定系統

## 基本公式

**攻擊力** = (體格×2 + 氣輸出×攻擊比率%) × 技法係數 + 1d10
**防禦力** = (耐久×1.5 + 氣輸出×防禦比率%) × 纏係數 + 1d6

攻擊比率/防禦比率由流（Ryu）的當前模式決定。預設為均衡（50/50）。

## 技法係數（攻擊）

| 技法 | 係數 | 說明 |
|------|------|------|
| 無念（純肉體）| ×0.5 | 完全不用氣 |
| 纏（Ten）| ×1.0 | 基礎念加護 |
| 練（Ren）| ×1.5 | 大量釋放氣 |
| 發（Hatsu）| ×2.0～3.0 | 依誓約強度決定 |
| 硬（Ko）| ×3.5 | 全力一擊，防禦歸零 |
| 周（Shu・武器）| ×1.4 | 氣纏繞在武器上 |
| 非擅長系技法 | ×0.6 | 使用其他念系的能力 |

## 纏的防禦係數

| 防禦姿態 | 係數 | 說明 |
|---------|------|------|
| 無纏 | ×0.5 | 完全暴露 |
| 基礎纏 | ×1.0 | 標準防禦 |
| 堅（Ken）| ×1.6 | 全身均勻念防禦 |
| 絕（Zetsu）| ×0 | 防禦歸零（但可隱匿存在）|

## 戰鬥結果判定

| 條件 | 結果 | HP 影響 |
|------|------|---------|
| 攻 > 防×2.0 | 壓倒性勝利 | 對方 HP -40% |
| 攻 > 防×1.5 | 明顯勝利 | 對方 HP -25%，己方 -5% |
| 攻 > 防 | 小勝 | 對方 HP -15%，己方 -10% |
| 攻 ≈ 防（±10%）| 互傷 | 雙方 HP -12% |
| 攻 < 防 | 失敗 | 己方 HP -20%，可撤退 |
| 攻 < 防×0.5 | 慘敗 | 己方 HP -40%，強制撤退 |

## 體力消耗

| 行動 | 消耗 |
|------|------|
| 一般移動 | -5 |
| 戰鬥一回合 | -10 |
| 使用練 | 額外 -5/回合 |
| 使用硬 | 額外 -15/次 |
| 使用發 | 氣消耗 × 0.1 |
| 休息一個行動 | +20 |
| 睡眠（過夜）| 完全恢復 |

體力歸零：攻防各 -30%。連續歸零可能導致昏迷。

## 先攻判定

先攻值 = 速度 + 凝等級加成（若已解鎖）+ 1d6
先攻值高者先行動。
流・高階（念 Lv.81+）：可在對方先攻瞬間反應切換流模式。

## 骰子機制

取當前時間戳（秒）的後兩位數字 mod 骰面數 + 1。
每次判定前向玩家展示計算過程：
「時間戳：...47，47 mod 10 + 1 = 8。你的骰子結果是 8。」

## 流（Ryu）模式

| 模式 | 攻 | 防 | 觸發語 |
|------|----|----|--------|
| 硬（Ko）| 100% | 0% | 「用硬」「出硬」|
| 猛攻 | 80% | 20% | 「猛攻」「全力進攻」|
| 均衡 | 50% | 50% | 預設/「均衡」|
| 堅（Ken）| 30% | 70% | 「堅」「切堅」|
| 全防 | 10% | 90% | 「全防」「死守」|

**流等級**：
| 等級 | 念等級 | 能力 |
|------|--------|------|
| 未解鎖 | < 51 | 固定均衡 |
| 初階 | 51-65 | 可切換，1 回合延遲 |
| 中階 | 66-80 | 即時切換 |
| 高階 | 81+ | 反應式切換 + 局部集中 |

**高階・局部集中**（念 Lv.81+）：
- 「流，硬集中右拳」→ 右拳攻擊最大化，其他部位防禦歸零
- 「流，堅護右肩」→ 右肩幾乎不受傷，整體反擊力極弱
