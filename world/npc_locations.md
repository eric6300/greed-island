# 貪婪之島 — 角色位置追蹤

## Phase 1：入島初期（Day 1-20）

| 角色 | 位置 |
|------|------|
| 小傑 + 奇犽 | 紫蘇樹 → 比諾特訓練 → 馬薩多拉 |
| 比司吉 | 紫蘇樹（加入小傑隊伍）|
| 西索 | 島內各處（以「庫洛洛」名義登錄）|
| 甘舒/沙度/巴拉 | 滲透尼克斯聯盟 |
| 俠客/小滴 /庫嗶 | 正規進入遊戲，收集卡牌 |
| 飛坦/芬克斯 | 島內各處，殺人奪牌 |
| 磊札 | 索夫拉比燈塔 |
| Eta | 紫蘇之樹 |

## Phase 2：聯盟滅亡期（Day 20-60）

| 角色 | 位置 |
|------|------|
| 奇犽 | 離開遊戲，參加獵人試驗（1 月 7 日）|
| 小傑 + 比司吉 | 訓練中，遇到阿本岡 |
| 甘舒 | 引爆炸彈後，前往Battera豪宅 |
| 阿本岡 | 進行除念儀式 |
| 幻影旅團（6人）| 坐船登島，被磊札驅至亞贊大陸 |
| 庫洛洛 | 更早被驅趕，此時在現實世界 |

## Phase 3：奇犽回歸（Day 60-90）

| 角色 | 位置 |
|------|------|
| 奇犽 | 通過試驗，重返遊戲 |
| 西索 | 愛愛，身份曝光 |
| 幻影旅團（正規入島後）| 各城市，監視阿本岡 |
| 絕茲絕拉 | 活動中，卡牌收集 |

## Phase 4：索夫拉比海盜戰（Day 90-120）

| 角色 | 位置 |
|------|------|
| 小傑隊 + 西索 + 果列奴 | 索夫拉比，挑戰磊札 |
| 磊札 | 索夫拉比燈塔 |
| 甘舒 | 追蹤絕茲絕拉的卡牌 |

## Phase 5：最終決戰（Day 120+）

| 角色 | 位置 |
|------|------|
| 甘舒 | 要求交出 一坪海岸線 |
| 絕茲絕拉 | 幫小傑拖延三週 |
| 小傑/奇犽/比司吉 | 秘密訓練 |
| 幻影旅團 | 取得除念師線索，逐漸淡出主線 |

## 玩家位置對 NPC 觸發的影響

- 玩家在**安托基巴**且為 15 日 → 月賽 NPC 出現
- 玩家在**索夫拉比**且首次 → Bopobo出現
- 玩家在**紫蘇樹**且比司吉未加入 → 比司吉出現
- 玩家 specified_count > 80 且在安托基巴 → 神秘買家出現
- 玩家念等級 > 25 且在愛愛 → 西索有機會接觸
