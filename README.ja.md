## 3つのコマンドで入島

```bash
git clone https://github.com/eric6300/greed-island
cd greed-island
claude
```

`/start` と入力。シソの木でイータが出迎えてくれる。

## これは何？

グリードアイランドは、**Claude Code の中だけで動くテキストアドベンチャーゲーム**だ。インストール不要、外部依存なし、プログラミング言語も不要。必要なのは `git` と [Claude Code](https://claude.ai/code) だけ。

Claude がゲームマスター（GM）として、没入感のあるテキストで世界全体を語る。島を探索し、カードを集め、念能力を鍛え、仲間を作るか敵を作るか——すべての選択がこの島の運命を変える。

---

## 特徴

### 原作に忠実なグリードアイランド

指定ポケットカード100枚、呪文カード40枚、12以上の都市と地域。アントキバの月例コンテストからスーフラビのドッジボール決勝、マサドラの呪文カードショップからドリアスの命がけのギャンブルまで——原作の全てがここにある。

### 完全な念能力システム

水見式で念系統を決定。纏・絶・練・発・凝・周・円・堅・硬・隠・流——11の念技術がレベルに応じて解放される。自分だけの必殺技を設計し、制約と力のバランスを見つけろ。

### ローグライク：死は永久

セーブロードなし。やり直しなし。HPがゼロになった瞬間、その周回は終わる。だが死は終わりではない——クリア（または死亡）のたびに新しい知識が解放され、次の周回でまったく異なる道が開ける。

### この島には意志がある

全てを変えられると思うか？やってみろ。

### NPC は覚えている

ビスケはトレーニングのたびにプレイヤーの努力を覚え、態度が軟化していく。ヒソカは闇から観察し、実力が上がるほど興味を示す。レイザーは正面から挑む者を尊敬する。ゲンスルー……はプレイヤーが先制攻撃に値するかを評価している。

これらの記憶は会話やセッションを越えて持続する。彼らは生きている。

### マルチ周回：毎回が新しい物語

クリア時に3枚のカードを持ち出せる——次の周回ではフルパワーで、遺産カードは100%の効果で再開。目撃した悲劇、逃したチャンス、間に合わなかった選択——次の周回では、何をすべきか分かっている。

クリアするたびに新しい可能性が解放される。失敗を知る者だけが見える道がある。

## コマンド

| コマンド | 説明 |
|---------|------|
| `/start` | 新しいゲームを開始、または前回の続きから |
| `/book` | 呪文の書を召喚——カード収集進捗を確認 |
| `/look` | 現在の場所を詳しく観察 |
| `/go [場所]` | 別の場所へ移動 |
| `/talk [NPC]` | キャラクターと会話 |
| `/gain` | 倒した敵や発見したアイテムをカード化 |
| `/cards` | 所持カード数をすばやく確認 |
| `/use [カード]` | カードの効果を発動 |
| `/transform` | カードを実体アイテムに変換 |
| `/slotin [カード]` | カードを指定ポケットに入れる |
| `/clone [カード] [仲間]` | カードを仲間に複製 |
| `/memcard` | メモリーカードを確認——クリア履歴、遺産、未解放エンディング |

**Book** や **Gain** と直接言うこともできる——GM はゲーム内の呪文として扱う。

## FAQ

**Q: HUNTER×HUNTER を知らなくてもプレイできる？**
A: できる。GM がゲーム中に全てのコンセプトを自然に説明してくれる。予備知識は不要。

**Q: 1周にどれくらいかかる？**
A: 探索の深さによる。メインストーリーだけなら数時間、フル探索ならもっと。

**Q: 本当にパーマデス？セーブロードなし？**
A: 本当だ。ただし知識と遺産カードは次の周回に引き継がれる。死ぬたびに強くなる。
