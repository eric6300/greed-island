## 3つのコマンドで入島

```bash
git clone https://github.com/eric6300/greed-island
cd greed-island
claude
```

`/start` と入力。あとは島が教えてくれる。

---

## これは何？

グリードアイランドは、**Claude Code の中だけで動くテキストアドベンチャーゲーム**だ。インストール不要、外部依存なし、プログラミング言語も不要。必要なのは `git` と [Claude Code](https://claude.ai/code) だけ。

Claude がゲームマスター（GM）として、没入感のあるテキストで世界全体を語る。島を探索し、カードを集め、念能力を鍛え、仲間を作るか敵を作るか——すべての選択には代償がある。

---

## 特徴

### 原作に忠実なグリードアイランド

都市、荒野、港、カジノ——原作の舞台がすべてここにある。カードは世界中に散らばり、それぞれに固有の入手条件がある。戦闘が必要なもの、知恵が必要なもの、思いもよらない代償を求めるもの。

### 完全な念能力システム

水見式で念系統を決定。冒険が進むにつれ、より強力な念技術が解放され、自分だけの必殺技を設計できる。制約が重いほど力は強い——だが代償も大きい。

### ローグライク：死は永久

セーブロードなし。やり直しなし。HPがゼロになった瞬間、すべて終わる。

だが、終わりは無駄を意味しない。

### この島には意志がある

全てを変えられると思うか？やってみろ。

### 彼らは生きている

島のキャラクターはお前を覚えている。お前のすべての行動が、彼らの目を変える。

## 基本コマンド

| コマンド | 説明 |
|---------|------|
| `/start` | 冒険を始める |
| `/look` | 周囲を観察する |
| `/go [場所]` | どこかへ移動する |
| `/talk [キャラ]` | 誰かと話す |
| `/book` | 呪文の書を開く |

他のコマンドはプレイ中に自然と覚える。**Book** や **Gain** と直接言うこともできる——GM はゲーム内の呪文として扱う。
